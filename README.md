# ファイルを上げるとき
0. code . でVSCodeを開く
1. ファイルを(例：txtフォルダの中に)ドラッグアンドドロップ
2. 左のバーの上から3つ目(ソース管理)を押す
3. コミットボタン→はいを押す
4. 何かメッセージを書いて、上のタブの×を押す
5. 保存しますか？に保存するを押す(左端)
6. 変更を同期ボタン→OK(左端)を押す
